What are ADA Accessible Documents

ADA Accessible Documents are compliant with Section 508 standards. In Other words, whether somebody has low vision, color blindness, difficulty with hearing, or any other disability, they can still receive all of the information from a document or video.

File Remediation Services are an important part of keeping all of you web content and files accessible.

What we check for

  • Color Contrast

    Making it easier for people with low vision to see and read with proper contrast.

  • Heading Structure

    Allows screen readers to accurately read the document

  • Image Alt text

    To help screen readers, every image needs an alt text describing it.

  • Reading Order

    Telling a screen reader the order in which to read a document.

  • Form Structure

    Using Form PDFs is common. Let’s make sure everybody can use them.

  • Other

    There are many items to check for. There are only 5 from a very long list.

What Documents must be accessible?

To be straightforward, every document needs to be accessible, if you want to meet the legal requirements at least. This includes, but is not limited to:

  • Word Documents
  • PDF Documents
  • Excel Sheets
  • PowerPoint Presentations
